---
description: 如何处理后端接口 404 问题
---

当您在前端遇到 `404 Not Found` 错误，且您确认 URL 拼写正确时，通常是因为后端服务没有更新。

1. **不要惊慌**：前端代码通常是实时生效的（HMR），但后端 Java 代码**必须重启**才能生效。
2. **操作步骤**：
   - 打开您的 IDE (IntelliJ IDEA)。
   - 找到正在运行的 Spring Boot 应用。
   - 点击 **Stop** (红色方块)。
   - 点击 **Rerun** (绿色运行按钮)。
3. **验证**：
   - 等待控制台输出启动成功的日志。
   - 在浏览器中刷新前端页面。
