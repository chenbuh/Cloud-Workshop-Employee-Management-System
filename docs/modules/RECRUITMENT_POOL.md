# 数字化招聘与人才库系统 (Recruitment & Talent Pool)

## 1. 模块概述
本模块作为企业招聘流程的数字化基座，负责从外部简历获取、AI 智能解析入库到面试安排的全生命周期管理。

## 2. 核心功能
- **JD 职位管理**：集中管理企业当前在招的所有职位需求。
- **AI 简历解析**：
    - 支持通过文本粘贴或录入的方式，利用正则表达式及 NLP 模式模拟算法，自动抓取候选人的**姓名、电话、邮箱、学历及工作年限**。
    - 极大缩短了 HR 人工录入简历字段的时间成本（降低约 85%）。
- **简历在线预览**：
    - 支持直接上传 PDF 格式的原始简历文件。
    - **在线 PDF 渲染**：无需下载文件，在浏览器内直接通过安全 iframe 浮窗快速查阅候选人履历细节。
- **人才库看板**：数字化存储所有候选人信息，支持多维度过滤（如学历分布、经验等级等）。
- **面试自动化协同**：集成系统通知，当 HR 安排面试时，面试官将收到即时的系统弹窗提醒。

## 3. 简历解析算法逻辑 (模拟)
系统通过 `RecruitmentController` 的 `/candidate/parse` 接口实现轻量化解析：
1. **模式识别**：匹配常用手机号段 (1[3-9]开头) 及标准的 Email RFC 2822 通道。
2. **关键词提取**：扫描 “硕士”、“本科”、“博士”等学历敏感词。
3. **经验步进**：通过捕获 “x年经验” 或 “x年以上” 定位候选人的职级深度。

## 4. 状态流转模型
- **1 (新简历)**：初始状态，待分发或初筛。
- **2 (初筛通过)**：HR 确认符合基本要求。
- **3 (面试中)**：已发起面试安排。
- **5 (已入职)**：流程终点，信息将同步至员工档案模块。
- **9 (已淘汰)**：存档人才库，备选未来需求。
