# 资产盘点与物资申领模块 (Asset Management)

## 1. 模块概述
本模块用于管理企业内部的固定资产（如电脑、桌椅）及办公办公耗材（如纸张、文具）。提供物资档案管理、申领流程审批及库存自动同步功能。

## 2. 核心功能
- **物资档案库**：分类展示所有行政资产，实时显示当前在库数量、规格及存放地点。
- **申领工作流**：
    1. 员工发起申领（选择物资、数量及用途）。
    2. 管理员在“申领进度”中进行审批。
    3. 审批通过后，系统自动扣减库存。
- **库存预警**：在视图顶部展示库存告急（少于5个）的物资项。
- **系统通知**：审批结果实时推送至员工通知中心。

## 3. 数据库设计
### 3.1 资产档案表 (`sys_asset`)
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGINT | 主键 |
| name | VARCHAR | 物资名称 |
| category | VARCHAR | 分类 (电脑, 耗材等) |
| stock | INT | 当前库存 |
| status | TINYINT | 1在库 0停用 |

### 3.2 申领记录表 (`sys_asset_claim`)
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGINT | 主键 |
| asset_id | BIGINT | 关联资产ID |
| user_id | BIGINT | 申领人ID |
| count | INT | 申领数量 |
| status | TINYINT | 0待审核 1通过 2驳回 |

## 4. 开发规范
- **事务一致性**：审批通过与库存扣减操作封装在同一个数据库事务 (`@Transactional`) 中。
- **UI 呈现**：采用 Tabs 切换架构，平衡资产展示与历史记录查询。
- **状态感知**：通过 Computed 属性实时计算待办任务数与库存状态。
