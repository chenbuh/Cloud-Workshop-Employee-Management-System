# 电子合同与在线预览模块 (Electronic Contract)

## 1. 模块概述
本模块实现了企业劳动合同及其补充协议的数字化管理。支持从起草、签署到归档的全流程线上化，并提供符合公文规范的在线预览与打印功能。

## 2. 核心功能
- **富文本起草**：集成 WangEditor，支持管理员在线编写复杂的合同条款。
- **在线预览**：
    - 模拟纸质公文排版，自动生成标题页与签署区。
    - **虚拟印章**：针对已生效合同，系统自动在签署位置覆盖红色“已签署”虚拟印章，增强视觉合规感。
- **打印与导出**：支持一键唤起浏览器打印控件，可直接保存为 PDF 电子档。
- **状态追踪**：详细记录合同的生效日期与失效日期，支持草稿、有效、过期等多种状态切换。

## 3. 数据库结构 (`sys_contract`)
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGINT | 主键 |
| employee_id | BIGINT | 签署员工ID |
| title | VARCHAR | 合同标题 |
| status | TINYINT | 状态 (0草稿 1有效 2过期) |
| content | LONGTEXT | 合同正文 (存储为 HTML) |
| start_date | DATE | 合同生效日 |

## 4. 技术特性
- **打印样式隔离**：通过在新窗口写入独立的 CSS 样式表，确保打印出的纸质合同格式整齐，不包含 UI 边框。
- **动态印章**：使用 CSS 旋转与边框技术纯手工绘制虚拟印章。
