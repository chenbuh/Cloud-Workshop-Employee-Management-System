# 流程引导系统 (Workflow Guidance System)

## 1. 模块概述
该模块专门用于处理员工在企业全生命周期中的关键节点：**入职 (Onboarding)** 与 **离职 (Offboarding)**。通过预设的标准动作列表，确保行政、HR、IT等部门的协同操作不遗漏、可追溯。

## 2. 核心功能
- **模板引擎**：支持为不同场景（如正式入职、实习生入职）定义不同的办理步骤。
- **任务分发**：启动流程后，系统根据模板自动拆解为多个分部任务，分配给对应的职能团队（HR/IT/行政）。
- **实时进度反馈**：看板式展示所有正在办理中的流程，通过百分比进度条直观体现当前任务完成情况。
- **闭环审计**：每个步骤的确认都需要实名操作并支持填写办理备注。

## 3. 标准流程节点
### 3.1 入职流程 (ONBOARDING)
1. **签署合同** (HR负责)
2. **分配硬件设备** (IT负责)
3. **门禁权限激活** (行政负责)
4. **文化导入培训** (行政负责)

### 3.2 离职流程 (OFFBOARDING)
1. **资产清退回收** (IT负责)
2. **账号权限注销** (IT负责)
3. **社保/五险停办** (HR负责)
4. **离职协议签署** (HR负责)

## 4. 技术方案
- **状态流转**：采用 `sys_flow_instance` (实例) 与 `sys_flow_task` (细分任务) 两级结构。
- **自动结项**：当所有细分任务标记为“已完成”时，系统自动将实例状态置为“已结束”。
- **UI交互**：利用侧边抽屉 (`Drawer`) 与时间轴 (`Timeline`) 深度集成，提供极佳的沉浸式操作体验。
