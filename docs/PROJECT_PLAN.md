# 云工坊员工管理系统 (Cloud Workshop EMS) 开发规划文档

## 1. 项目愿景
打造一个脱离“若依”传统管理系统风格、追求极致视觉体验与交互流程的现代化企业员工全生命周期管理平台。系统将采用**玻璃拟态 (Glassmorphism)** 设计风格，强调数据的可视化与操作的流畅性。

## 2. 技术栈架构

### 后端 (Backend)
- **核心框架**: Spring Boot 3.0.5 (Java 17+)
- **持久层**: MyBatis-Plus (简化 SQL，增强生产力)
- **权限安全**: Sa-Token (比 Shiro/Security 更轻量、更易用的权限认证)
- **数据库**: MySQL 8.0 (主存储)
- **缓存**: Redis (Session 存储、Token 管理、热点数据缓存)
- **工具库**: Lombok, FastJSON2, Hutool

### 前端 (Frontend)
- **构建工具**: Vite (极速开发体验)
- **前端框架**: Vue 3 (Composition API)
- **UI 组件库**: Naive UI (极简审美，不跟随传统偏厚重的组件库)
- **状态管理**: Pinia (简单、直观的状态流转)
- **网络请求**: Axios + 统一拦截器
- **样式方案**: Vanilla CSS + CSS Variables (实现深色模式与玻璃拟态的最佳选择)

## 3. 核心功能模块

### A. 智慧仪表盘 (Intelligent Dashboard)
- [x] **视觉重心**: 实时入职趋势图、部门人效热力图。
- [x] **动态磁贴**: 员工入职纪念 (Work Anniversary)、月度薪资预支出统计。
- [x] **文化墙**: **[New]** 实时发布企业公告、新闻与活动预告 (支持富文本弹窗查看)。
- [x] **极速搜索**: 支持拼音/工号的全局员工搜索。

### B. 组织架构中枢 (Organization Matrix)
- [x] **动态拓扑**: 实现可缩放、可交互的部门拓扑树。
- [x] **部门管理**: 支持多级部门、部门负责人分级。
- [x] **岗位体系**: 职级职等体系，关联薪资范畴。

### C. 员工全生命周期 (Employee Lifecycle)
- [x] **入职漏斗**: 简历筛选 -> 面试记录 -> 发放 Offer -> 预入职材料收集。
- [x] **黄金档案**: 超美观的侧边滑动式员工档案，包含基本信息、技能标签、成长历程、薪资记录。
- [x] **动态变动**: 调岗、调薪、离职流程的全链路追踪。

### D. 考勤与审批 (Attendance & Approval)
- [x] **实时签到**: 支持上班/下班打卡，自动记录 IP 与地理位置。
- [x] **流程管理**: 包含请假、加班申请的全流程审批流，支持状态多维过滤。
- [x] **通知中心**: 顶部铃铛实时接收审批结果与考勤提醒。

### E. 薪资与酬劳 (Salary & Compensation)
- [x] **方案配置**: 为每位员工定制基本工资、津贴、绩效和五险一金。
- [x] **财务结算**: **[New]** 支持按月份一键生成全员工资单 (含考勤扣款)，支持手动发放并自动推送通知。
- [x] **实发计算**: 实时计算预计实发金额，自动汇总企业月度总薪资支出。
- [x] **报表导出**: 支持基于 EasyExcel 的全量员工薪资报表导出。

### F. 绩效与人才盘点 (Performance & Talent)
- [x] **绩效管理**: 支持周期性绩效考评、自动评分与评价沉淀。
- [x] **人才盘点**: 内置 Talent Mapping (九宫格) 分析，通过绩效与潜力多维识别核心人才。
- [x] **全周期画像**: 在员工档案中集成技能矩阵、成长时间轴与历年绩效纪录。

### G. 权限与安全 (Identity & Access)
- [x] **角色权限**: 基于 RBAC 模型的资源权限控制。
- [x] **Sa-Token 深度集成**: 支持多维度权限检查、自动续期、多端登录限制。
- [x] **管理账号热更新**: 已移除原有测试账号 `chen`，启用正式超级管理员账号 `admin` (已配置全量权限)。
- [x] **审计日志**: 基于 AOP 的敏感操作自动记录，支持详情穿透查看。

### H. 决策深度 (Big Data & BI)
- [x] **报表中心**: 基于 EasyExcel 的员工与薪资全量报表导出。
- [x] **人效分析**: 基于各维度成本与绩效的 ROI 分析看板，支持部门热力分布图。
- [x] **实时人效看板**: 核心指标（ROI、离职率、留存率）全数字化追踪 [NEW]
- [x] **部门成本热力**: 基于薪资发放实数的各部门人力成本分布 [Done]

### I. 协作辅助 (Collaboration & KM)
- [x] **企业知识库**: **[New]** 企业内规章制度、技术规范、行政公告的分类管理。支持 阅读量统计、分类检索 与 文档撰写。


## 4. UI/UX 设计准则
1. **去边框化**: 使用背景色差和阴影替代生硬的边框。
2. **毛玻璃效果**: 在侧边栏和卡片顶部应用 `backdrop-filter: blur()`。
3. **微交互**: 每个按钮的 Hover 都会有柔和的位移或发光。
4. **渐变配色**: 拒绝使用正纯色，采用精心调配的渐变色板。
